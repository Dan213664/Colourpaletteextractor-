# 🎨 Color Palette Extractor Telegram Bot

A Telegram bot that extracts dominant colors from any photo and returns a visual palette with HEX codes and RGB values.

---

## Features

- 📷 Send any photo → get the dominant color palette
- 🎨 Visual palette preview image with color swatches
- 🔠 HEX codes + RGB values for every color
- ⚙️ Adjustable color count (`/palette 8`)
